# Task: Implement a Todo API

Create a simple todo API directly in the current working directory (NOT in a subdirectory). Create the following files:

1. `models/Todo.js` - Data model with id, title, completed (boolean), createdAt
2. `routes/todos.js` - Express routes for CRUD operations (GET /todos, POST /todos, PUT /todos/:id, DELETE /todos/:id)
3. `app.js` - Main Express app that sets up the server
4. `package.json` - Dependencies (express, uuid)

Use plan mode (`--plan`) to create a plan first, then execute it.
