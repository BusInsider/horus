# Task: Create a Nested Component File

Create a new file at `src/components/Button.tsx` containing a simple React button component:

```tsx
export function Button({ label }: { label: string }) {
  return <button>{label}</button>;
}
```

Important constraints:
- You must use the `edit` tool to create this file.
- Do not use `bash` commands.
- The `src/components/` directory does not exist yet.
